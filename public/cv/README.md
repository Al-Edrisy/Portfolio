# CV File Location

Place your CV file here with the name: `Salih_Ben_Otman_CV.pdf`

The download button on the `/skills` page will download this file.

**File path:** `/public/cv/Salih_Ben_Otman_CV.pdf`

**To add your CV:**
1. Place your PDF CV file in this directory
2. Rename it to `Salih_Ben_Otman_CV.pdf`
3. The download button will automatically work

**Alternative:** If you want to use a different filename, update the path in:
- `components/sections/skills-section.tsx` (line ~295)
- Change `link.href = '/cv/YOUR_FILENAME.pdf'`
